Інформаційна сторінка про атрибут media

Назва атрибуту: < media >

Опис призначення:

Атрибут media Встановлює пристрій виведення, для якого призначена таблиця стилів. Для кожного пристрою - від кишенькового комп'ютера до принтера можна визначити свій власний стиль, який би враховував специфіку пристрою і підганяв під нього вид веб-сторінки.

Синтаксис:

< style media="all|braille|handheld|print|screen|speech|projection|tty|tv">

...

</style>

Значення:

all -Всі пристрої.
braille-Улаштування, засновані на системі Брайля, призначені для сліпих людей.
handheld-Наладоннике, смартфони, пристрої з малою шириною екрану.
print-Друкувальний пристрій на зразок принтера.
screen-Єкран монітора
speech-Мовні синтезатори, а також програми для відтворення тексту вголос. Сюди ж входять мовні браузери.
projection-Проектор.
tty-Телетайпи, термінали, портативні пристрої з обмеженими можливостями екрану. Для них не повинні використовуватися пікселі як одиниці виміру.
tv-Телевізор.

Значення по замовчуванню:

screen
